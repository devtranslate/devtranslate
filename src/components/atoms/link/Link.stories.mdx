# Link

O componente **Link** representa um conteúdo clicável que executa alguma ação, geralmente levando o usuário a outro contexto.

<Canvas>
  <Story name="Default" args={{ children: 'Ir para a página inicial', href: 'https://devtranslate.io/' }}>

  </Story>
</Canvas>

## Tamanhos

Com a propriedade **size**, é possível configurar o componente em três tamanhos diferentes:

export const linkSizes = ['l', 'm', 's'];

<Story name="Size">
  <View inline={{ xs: false }} paddingTop={{ xs: 'xs' }} paddingBottom={{ xs: 'm' }}>
    {linkSizes.map((item, index) => {
      return (
        <Link
          size={{ xs: item }}
          paddingBottom={{ xs: 'xs' }}
          onClick={() => alert('Exemplo de onClick do Link!')}
          key={`text-${index - 1}`}
        >
          Exemplo de Link {item}
        </Link>
      );
    })}
  </View>
</Story>

## Alinhamento

Utilize a propriedade **textAlign** para alterar o alinhamento do texto.

export const textAlignment = ['left', 'center', 'right'];

</Story>
