O componente **Text** serve para criar parágrafos e textos no geral.

export const TextExample =
  'Lorem ipsum dolor sit amet, consectetur adipiscing elit, sed do eiusmod tempor incididunt ut labore et dolore magna aliqua. Ut enim ad minim veniam, quis nostrud exercitation ullamco laboris nisi ut aliquip ex ea commodo consequat. Duis aute irure dolor in reprehenderit in voluptate velit esse cillum dolore eu fugiat nulla pariatur. Excepteur sint occaecat cupidatat non proident, sunt in culpa qui officia deserunt mollit anim id est laborum.';

## Tamanhos

Com a propriedade **size**, é possível configurar o componente em três tamanhos diferentes:

export const textSizes = ['xl', 'l', 'm', 's', 'xs'];

<Story name="Size">
  <View inline={{ xs: false }} paddingTop={{ xs: 'xs' }} paddingBottom={{ xs: 'm' }}>
    {textSizes.map((item, index) => {
      return (
        <Text size={{ xs: item }} paddingBottom={{ xs: 'xs' }} key={`text-${index - 1}`}>
          Exemplo de Text {item}
        </Text>
      );
    })}
  </View>
</Story>

export const textColors = ['', 'primary', 'secondary', 'tertiary'];

## Capitalização

Utilizando a propriedade **textTransform** é possível mudar a capitalização do componente.

export const textCapitalizations = ['none', 'capitalize', 'uppercase', 'lowercase'];

<Story name="TextTransform">

    {textCapitalizations.map((item, index) => {

## Semântica

Atualmente este componente oferece as tags `b`, `i`, `p`, `s`, `small`, `span`, `strong` e `u`. Por meio de sua utilização o componente será exibido de forma diferente em cada caso, como é possível observar abaixo:

export const textTags = [
  { tag: 'b', text: 'bold' },
  { tag: 'i', text: 'italic' },
  { tag: 'p', text: 'paragraph' },
  { tag: 's', text: 'strikethrough' },
  { tag: 'small', text: 'small' },
  { tag: 'span', text: 'generic' },
  { tag: 'strong', text: 'strong' },
  { tag: 'u', text: 'underline' },
];

<Story name="Tag">
  <View inline={{ xs: false }} paddingTop={{ xs: 'xs' }} paddingBottom={{ xs: 'm' }}>
    {textTags.map((item, index) => {
      return (
        <Text tag={item.tag} paddingBottom={{ xs: 'xs' }} key={`text-${index - 1}`}>
          Exemplo de Text ({item.text})
        </Text>
      );
    })}
  </View>
</Story>

> Lembre-se sempre de analizar o componente que pretende implementar e passar a prop `tag` condizente com o contexto onde ele será inserido. Esta prática é importante pois ajuda a prezar pela semântica da página.
