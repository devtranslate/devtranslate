# Colors

A paleta de cores base totaliza 92 cores, incluindo 9 cores primárias e suas cores derivadas.

<br />

### Acessibilidade

Temos o compromisso de cumprir as **Diretrizes de Acessibilidade para Conteúdo Web (WCAG)** com taxas de contraste mínimo em nível AA. Desta forma use cores focadas e significativas para identificar exatamente o que as pessoas precisam ver, garantindo contraste suficiente entre os elementos para que pessoas com baixa visão possam ver e usar nossos produtos.

<br />

## Marca

Nossa paleta é composta por rosa, amarelo e azul-acizentado para trazer ousadia à nossa marca e é usada de maneira lógica em todo o produto e marketing para guiar o olhar e destacar as partes importantes.

export const renderColorItem = (color, colorName, colorShade) => {
  return (
    <Grid>
      <ColorItem color={color} />
      <View inline={{ xs: false }} paddingLeft={{ xs: 'xs' }}>
        <Text as="b" textTransform="capitalize">
          {colorName} {colorShade ? colorShade : ''}
        </Text>
        <Text>{color}</Text>
      </View>
    </Grid>
  );
};

export const colorShades = [50, 100, 200, 300, 400, 500, 600, 700, 800, 900];
